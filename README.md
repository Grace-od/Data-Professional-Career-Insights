# Data Professional Survey Breakdown

## 📊 Project Overview
This Power BI project provides a detailed breakdown of survey results from 630 data professionals globally. It explores the relationship between job titles, average salaries, and workplace sentiment, while also tracking the most popular programming languages in the industry today.

## 🚀 Key Features & Analysis
* **Demographic Breakdown:** Analyzed data from 630 survey takers with an average age of 29.87.
* **Salary Insights:** Visualized average salary by job title, highlighting the earning potential across different data roles.
* **Workplace Sentiment:** Developed gauge charts to track "Happiness with Salary" (4.27/10) and "Work-Life Balance" (5.74/10).
* **Technical Landscape:** Identified Python as the leading favorite programming language among survey participants.
* **Geographic Distribution:** Utilized Treemaps to show the concentration of professionals across the US, India, UK, Canada, and other regions.
* **Industry Barriers:** Analyzed the perceived difficulty of breaking into the data field.

## 🛠️ Technical Implementation
* **Advanced DAX:** Implemented complex calculations to drive the dashboard's metrics and gauge visuals.
* **Data Transformation:** Cleaned and structured the "Power BI - Final Project.xlsx" dataset for accurate reporting.
* **Interactive Design:** Created a summary report featuring bar charts, donut charts, and treemaps for intuitive user navigation.

## 📂 Project Structure
* `PowerBI_Week10_YourName.pbix`: The final interactive Power BI report file.
* `Power BI – Mini Dashboard Project.xlsx`: The source dataset used for the analysis.

## 📖 How to View
1. Download the `.pbix` file from this repository.
2. Open the file using [Power BI Desktop](https://powerbi.microsoft.com/desktop/).
3. Explore the various visualizations to gain insights into the data professional career path.
